coordsClean

  • coordsClean(feature: Feature | GeometryObject): void
    • Defined in modify.ts:28

    Removes redundant coordinates from any GeoJSON Geometry.

    example

    var line = geo.create.lineString([[0, 0], [0, 2], [0, 5], [0, 8], [0, 8], [0, 10]]); var multiPoint = geo.create.multiPoint([[0, 0], [0, 0], [2, 2]]);

    geo.coords.cleanCoords(line).geometry.coordinates; //= [[0, 0], [0, 10]]

    geo.coords.cleanCoords(multiPoint).geometry.coordinates; //= [[0, 0], [2, 2]]

    Parameters

    • feature: Feature | GeometryObject

      Feature or Geometry

    Returns void

    the cleaned input Feature/Geometry

pointsClusterKmeans

  • pointsClusterKmeans(points: FeatureCollection<Point>, num: number): void
    • Defined in modify.ts:112

    Takes a set of Point|points and partition them into clusters using the k-mean . It uses the k-means algorithm

    example

    // create random points with random z-values in their properties var points = geo.random.point(100, {bbox: [0, 30, 20, 50]}); var options = {numberOfClusters: 7}; var clustered = geo.cluster.kmeans(points, options);

    Parameters

    • points: FeatureCollection<Point>

      to be clustered

    • num: number

      number of clusters that will be generated

    Returns void

    Clustered Points with an additional two properties associated to each Feature:

    • {number} cluster - the associated clusterId
    • {[number, number]} centroid - Centroid of the cluster [Longitude, Latitude]

transformRotate

  • transformRotate(features: turf.AllGeoJSON, angle: number, pivot: Point): void
    • Defined in modify.ts:219

    Rotates any geojson Feature or Geometry of a specified angle, around its centroid or a given pivot point; all rotations follow the [right-hand rule] (https://en.wikipedia.org/wiki/Right-hand_rule)

    example

    var poly = geo.create.polygon([[[0,29],[3.5,29],[2.5,32],[0,29]]]); var options = {pivot: [0, 25]}; var rotatedPoly = geo.xform.transformRotate(poly, 10, options);

    Parameters

    • features: turf.AllGeoJSON

      object to be rotated

    • angle: number

      of rotation (along the vertical axis), from North in decimal degrees, negative clockwise

    • pivot: Point

      point around which the rotation will be performed

    Returns void

    the rotated GeoJSON feature
