contains

  • contains(feature1: Feature, feature2: Feature): boolean
    • Defined in old/compare.ts:28

    Returns True if the second geometry is completely contained by the first geometry. The interiors of both geometries must intersect and, the interior and boundary of the secondary (geometry b) must not intersect the exterior of the primary (geometry a). (Opposite result of within.)

    example

    var line = geo.create.lineString([[1, 1], [1, 2], [1, 3], [1, 4]]); var point = geo.create.point([1, 2]);

    geo.compare.contains(line, point); //=true

    Parameters

    • feature1: Feature

      GeoJSON Feature or Geometry

    • feature2: Feature

      GeoJSON Feature or Geometry

    Returns boolean

    true/false

crosses

  • crosses(feature1: Feature, feature2: Feature): boolean
    • Defined in old/compare.ts:49

    Returns true if the intersection results in a geometry whose dimension is one less than the maximum dimension of the two source geometries and the intersection set is interior to both source geometries.

    Returns true for only multipoint/polygon, multipoint/linestring, linestring/linestring, linestring/polygon, and linestring/multipolygon comparisons.

    example

    var line1 = geo.create.lineString([[-2, 2], [4, 2]]); var line2 = geo.create.lineString([[1, 1], [1, 2], [1, 3], [1, 4]]);

    var cross = geo.compare.crosses(line1, line2); //=true

    Parameters

    • feature1: Feature

      GeoJSON Feature or Geometry

    • feature2: Feature

      GeoJSON Feature or Geometry

    Returns boolean

    true/false

pointInPolygon

  • pointInPolygon(point: Point, polygon: Polygon, ignoreBoundary: boolean): boolean
    • Defined in old/compare.ts:114

    Takes a Point and a Polygon or MultiPolygon and determines if the point resides inside the polygon. The polygon can be convex or concave. The function accounts for holes.

    example

    var pt = geo.create.point([-77, 44]); var poly = geo.create.polygon([[ [-81, 41], [-81, 47], [-72, 47], [-72, 41], [-81, 41] ]]);

    geo.compare.pointInPolygon(pt, poly); //= true

    Parameters

    • point: Point

      input point

    • polygon: Polygon

      input polygon or multipolygon

    • ignoreBoundary: boolean

      True if polygon boundary should be ignored when determining if the point is inside the polygon otherwise false.

    Returns boolean

    true if the Point is inside the Polygon; false if the Point is not inside the Polygon
