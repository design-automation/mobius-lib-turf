cleanCoords

  • cleanCoords(geojson: any, parameters: any): any
    • Defined in coords.ts:29

    Removes redundant coordinates from any GeoJSON Geometry.

    example

    var line = turf.lineString([[0, 0], [0, 2], [0, 5], [0, 8], [0, 8], [0, 10]]); var multiPoint = turf.multiPoint([[0, 0], [0, 0], [2, 2]]);

    turf.cleanCoords(line).geometry.coordinates; //= [[0, 0], [0, 10]]

    turf.cleanCoords(multiPoint).geometry.coordinates; //= [[0, 0], [2, 2]]

    Parameters

    • geojson: any

      Feature or Geometry

    • parameters: any

    Returns any

    the cleaned input Feature/Geometry
