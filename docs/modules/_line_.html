slice

  • slice(startPt: any, stopPt: any, line: any): Feature<LineString, object>
    • Defined in line.ts:165

    Takes aLineString, a start Point, and a stop point and returns a subsection of the line in-between those points. The start & stop points don't need to fall exactly on the line.

    This can be useful for extracting only the part of a route between waypoints.

    example

    var line = turf.lineString([ [-77.031669, 38.878605], [-77.029609, 38.881946], [-77.020339, 38.884084], [-77.025661, 38.885821], [-77.021884, 38.889563], [-77.019824, 38.892368] ]); var start = turf.point([-77.029609, 38.881946]); var stop = turf.point([-77.021884, 38.889563]);

    var sliced = turf.lineSlice(start, stop, line);

    Parameters

    • startPt: any

      starting point

    • stopPt: any

      stopping point

    • line: any

      line to slice

    Returns Feature<LineString, object>

    sliced line
