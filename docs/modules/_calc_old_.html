    • point: Point

voronoi

  • voronoi(points: FeatureCollection<Point>, bbox: turf.BBox): FeatureCollection<Polygon>
    • Defined in calc-old.ts:402

    The Voronoi algorithim used comes from the d3-voronoi package.

    example

    Parameters

    • points: FeatureCollection<Point>

      to find the Voronoi polygons around.

    • bbox: turf.BBox

      Optional clipping rectangle, in [minX, minY, maxX, MaxY] order.

    Returns FeatureCollection<Polygon>

    a set of polygons, one per input point.
