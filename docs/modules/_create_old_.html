arc

  • arc(center: Point, radius: number, bearing1: number, bearing2: number, steps: number): Feature<LineString>
    • Defined in create-old.ts:32

    Creates a circular arc, of a circle of the given radius and center point, between bearing1 and bearing2; 0 bearing is North of center point, positive clockwise.

    example

    var center = geo.create.point([-75, 40]); var radius = 5; var bearing1 = 25; var bearing2 = 47;

    var arc = geo.create.arc(center, radius, bearing1, bearing2);

    Parameters

    • center: Point

      center point

    • radius: number

      radius of the circle in meters

    • bearing1: number

      angle, in decimal degrees, of the first radius of the arc

    • bearing2: number

      angle, in decimal degrees, of the second radius of the arc

    • steps: number

      number of steps

    Returns Feature<LineString>

    line arc

bezierSpline

  • bezierSpline(line: LineString, resolution: number, sharpness: number): Feature<LineString>
    • Defined in create-old.ts:76

    Takes a LineString and returns a curved version by applying a Bezier spline algorithm.

    The bezier spline implementation is by Leszek Rybicki.

    example

    var line = geo.create.lineString([ [-76.091308, 18.427501], [-76.695556, 18.729501], [-76.552734, 19.40443], [-74.61914, 19.134789], [-73.652343, 20.07657], [-73.157958, 20.210656] ]);

    var curved = geo.line.bezierSpline(line);

    Parameters

    • line: LineString

      input LineString

    • resolution: number

      time in milliseconds between points

    • sharpness: number

      a measure of how curvy the path should be between splines

    Returns Feature<LineString>

    curved line
