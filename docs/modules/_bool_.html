difference

  • difference(poly1: any, poly2: any): Feature<Polygon | MultiPolygon, object>
    • Defined in bool.ts:42

    Finds the difference between two polygons by clipping the second polygon from the first.

    example

    var polygon1 = turf.polygon([[ [128, -26], [141, -26], [141, -21], [128, -21], [128, -26] ]], { "fill": "#F00", "fill-opacity": 0.1 }); var polygon2 = turf.polygon([[ [126, -28], [140, -28], [140, -20], [126, -20], [126, -28] ]], { "fill": "#00F", "fill-opacity": 0.1 });

    var difference = turf.difference(polygon1, polygon2);

    Parameters

    • poly1: any

      input Polygon feature

    • poly2: any

      Polygon feature to difference from polygon1

    Returns Feature<Polygon | MultiPolygon, object>

    a Polygon or MultiPolygon feature showing the area of polygon1 excluding the area of polygon2 (if empty returns null)

intersect

  • intersect(poly1: any, poly2: any): Feature<any, object>
    • Defined in bool.ts:75

    Takes two Polygon and finds their intersection. If they share a border, returns the border; if they don't intersect, returns undefined.

    example

    var poly1 = turf.polygon([[ [-122.801742, 45.48565], [-122.801742, 45.60491], [-122.584762, 45.60491], [-122.584762, 45.48565], [-122.801742, 45.48565] ]]);

    var poly2 = turf.polygon([[ [-122.520217, 45.535693], [-122.64038, 45.553967], [-122.720031, 45.526554], [-122.669906, 45.507309], [-122.723464, 45.446643], [-122.532577, 45.408574], [-122.487258, 45.477466], [-122.520217, 45.535693] ]]);

    var intersection = turf.intersect(poly1, poly2);

    Parameters

    • poly1: any

      the first polygon

    • poly2: any

      the second polygon

    Returns Feature<any, object>

    returns a feature representing the point(s) they share (in case of a Point or MultiPoint), the borders they share (in case of a LineString or a MultiLineString), the area they share (in case of Polygon or MultiPolygon). If they do not share any point, returns null.

union

  • union(polys: any): Feature<Polygon | MultiPolygon, object>
    • Defined in bool.ts:103

    Takes two or more Polygon and returns a combined polygon. If the input polygons are not contiguous, this function returns a MultiPolygon feature.

    example

    var poly1 = turf.polygon([[ [-82.574787, 35.594087], [-82.574787, 35.615581], [-82.545261, 35.615581], [-82.545261, 35.594087], [-82.574787, 35.594087] ]], {"fill": "#0f0"}); var poly2 = turf.polygon([[ [-82.560024, 35.585153], [-82.560024, 35.602602], [-82.52964, 35.602602], [-82.52964, 35.585153], [-82.560024, 35.585153] ]], {"fill": "#00f"});

    var union = turf.union([poly1, poly2]);

    Parameters

    • polys: any

      An array of polygons to combine

    Returns Feature<Polygon | MultiPolygon, object>

    a combined Polygon or MultiPolygon feature
